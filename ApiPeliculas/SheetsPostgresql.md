| Comando        | Función                           |
| -------------- | --------------------------------- |
| `\l`           | Listar bases de datos             |
| `\c nombre_db` | Conectarse a una base de datos    |
| `\conninfo`    | Ver conexión actual               |
| `\dn`          | Listar esquemas                   |
| `\dt`          | Listar tablas                     |
| `\dt schema.*` | Listar tablas de un esquema       |
| `\dv`          | Listar vistas                     |
| `\ds`          | Listar secuencias                 |
| `\di`          | Listar índices                    |
| `\d tabla`     | Ver estructura de una tabla       |
| `\d+ tabla`    | Ver estructura detallada          |
| `\df`          | Listar funciones                  |
| `\df+`         | Funciones con detalle             |
| `\x`           | Toggle vista expandida (vertical) |
| `\timing`      | Mostrar tiempo de ejecución       |
| `\! cls`       | Limpiar pantalla (Windows)        |
| `\q`           | Salir de `psql`                   |
